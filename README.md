# visual-format-language

-tips-

H --> HorizontalView
V --> VerticalView
| --> superView
- --> standardSpace
== --> equalHeigth or equlaWidth (depending of the orientation H or V)
>= --> we can use greater or equal and less than or equal
@ --> set priority ex:
	[orangeView(>=150@250)]
